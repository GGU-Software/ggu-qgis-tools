# Resources

This folder contains icon files for the plugin toolbar.

## Required Icons

Create or add the following PNG icons (recommended size: 24x24 or 32x32):

- `icon.png` - Main plugin icon (shown in QGIS plugin manager)
- `icon_open.png` - "Open in GGU-STRATIG" action
- `icon_create_borehole.png` - "Create Borehole" action
- `icon_create_sounding.png` - "Create Sounding" action
- `icon_settings.png` - "Settings" action

## Temporary Workaround

Until proper icons are added, the plugin will show default icons or no icons.
The functionality will still work.
